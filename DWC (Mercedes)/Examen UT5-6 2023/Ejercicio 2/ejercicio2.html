Crea un formulario con un campo de texto para introducir el código de un producto. El código debe cumplir obligatoriamente el siguiente formato: tres letras, guión y de 1 a 5 números (ej: ASX-23). Si el campo no es correcto, debe mostrar al salir del campo un mensaje que informe al usuario. El formulario también debe tener un botón para ser enviado, pero este no se enviará si el código no se ha introducido correctamente. Además en el momento en que se envía el formulario, el código debe ser almacenado en el navegador de forma permanente sin necesidad de que pase por el servidor.
